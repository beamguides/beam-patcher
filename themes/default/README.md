# Default Theme

The basic Beam Patcher theme with minimal customization.

## Description

This is the default theme that comes with Beam Patcher. It features a clean, simple design that works well for any Ragnarok Online server. Use this as a starting point for creating your own custom theme.

## Features

- ✅ Clean and minimal design
- ✅ Works on all screen resolutions
- ✅ Lightweight (no video background)
- ✅ Easy to customize

## Installation

1. Copy `config.yml` to your patcher root directory
2. Copy `assets/` folder to your patcher root directory
3. Run `beam-patcher.exe`

## Customization

Edit `config.yml` to customize:
- Server name and window title
- Logo and background image
- Button labels and URLs
- Colors and layout

## Assets Included

- `assets/logo.png` - Default logo (200x200px)
- `assets/background.jpg` - Default background (1920x1080px)
- `assets/icons/button-website.png` - Website button icon
- `assets/icons/button-forum.png` - Forum button icon
- `assets/icons/button-discord.png` - Discord button icon

## Technical Details

**Resolution:** 800x600 minimum, 1920x1080 recommended  
**File Size:** ~500KB total  
**Video Background:** No  
**Background Music:** No
